## 还在用 Excel + 微信群管工厂？

> 项目一张表、人员一张表、备品又一张表 — **改一处，对三处。**  
> 群里喊完领料、换人、延期，**没人能 30 秒答领导：哪条线最缺人、哪仓最低库存。**  
> 月底盘点通宵对账，**领用没流水，库存靠猜。**

**如果你点头了，这套系统就是为你做的。**

## 雷霆智能是什么？

**不是又一个填表网站。**  
是把 **指挥舱 · 产线节点 · 人员矩阵 · 备品流水 · 两级盘点 · 审批邮件 · 财务 TCO · 雷霆AI** 收进 **同一套平台** 的制造管控系统。

Web + 手机 H5 · 多厂区数据隔离 · 四级权限 · 全链路可审计 —  
从 **领用扣库存** 到 **现场拍照盘点** 再到 **辖区初审 · 系统终审**，制度化的标准流程，不是月底突击对账。

## 凭什么让你想马上找我？

### 📦 备品：流水驱动库存，不是期末猜数

入库 · 领用 · 借调 · 归还 · 报废 — **每一笔可查、可追责。**  
两级盘点：现场拍照 → 初审 → 终审 → 归档，**审计要材料时直接导出。**  
雷霆AI 盯低库存、呆滞、用量异常，**自动推管理员。**

### ⚙️ 产线：节点看得见，延期看得见

里程碑 1.1–1.11、计划/实际日期、白班/夜班编制 — **甘特一眼全局。**  
工程师 **≥3 个项目自动标超载**，项目经理不靠记忆排人。

### 👥 人员：谁在哪条线，系统说了算

正式工 + 外协、矩阵负荷、派驻工时；调配走 **审批 + 邮件**，通过/驳回有记录，**告别「群里喊完就忘」。**

### 🎯 指挥舱：早会前 30 秒，全局已就绪

项目健康度、人员负荷、预算执行、雷霆AI 预警 **一屏汇总** — 今天该盯什么，进门就知道。

### 💰 财务：看见真成本

全成本 TCO = 物料 + 影子人力 + 外协工时；薪资按工日 **摊到项目** — 项目赚不赚钱，用数字说话。

### 🤖 雷霆AI：厂内顾问，不是聊天机器人

组织诊断 · 预算人手联合分析 · 甘特自然语言改期 · 备品预警 — **冷静、可执行、可落地。**
